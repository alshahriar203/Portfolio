Connect4

This was my University 1st Year Project: Connect 4 in C. This game uses a Command Line Interface to play the game. It also features an AI adversary which is implemented in a IDS Minimax Algorithm. The entire game was written in C with no external Libraries used.

Mahir Chess

This was my University 2nd Year Project: Mahir Chess in Java. This features a 2 player Chess Game, written completely in Java, and AWT framework as the minimalist GUI. This show case various design patterns, and OOP principles and designs used in games.

PSD Graphics/Text Remover

This is a Command Line Utility that does Batch processing on PSD files, and converts them to png files with 1. Text Elements Removed, 2. Graphics Elements Removed. This is a handy python utility created mainly for the usage of Graphics Designers. The psd-tools library is used.
